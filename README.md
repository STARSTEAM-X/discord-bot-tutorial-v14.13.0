# discord-bot-tutorial-v1.0

### Installation
create config.json near index.js
 ```json
{
  "token": "",
  "MongoUrl": "",
  "clientId": "",
  "guildId": "",
  "ownerId": ""
}
```

install

```
npm install discord.js@14.13.0
npm install mongoose@7.6.0
```
